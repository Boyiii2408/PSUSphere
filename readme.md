# PSUSphere

### Project Description
PSUSphere is a Django-based web application designed to manage student organizations, academic programs, and college structures. It serves as a centralized platform to track student memberships and organizational activities within the university ecosystem.

### Key Features
* **College & Program Management**: Organized database of university colleges and their respective degree programs (BS and BA formats).
* **Organization Tracking**: Management of student-led organizations like ACS and SITE.
* **Student Directory**: A comprehensive list of students and their academic affiliations.
* **Membership Records**: Tracking of student enrollments in various campus organizations.
* **Refactored Admin Interface**: Customized Django Admin with advanced filtering and search capabilities for efficient data management.
